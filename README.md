# ST - Prosjektoppgave

Prosjektoppgave i emnet Skytjenester ved Fagskolen Innlandet.

Oppgaven tar utgangspunkt i en eksisterende on-prem løsning fra det tverrfaglige prosjektet og viser hvordan løsningen kan planlegges og delvis implementeres i Azure.

## Innhold
- Analyse av eksisterende løsning
- Forslag til Azure-arkitektur
- Migreringsstrategi
- Risikoanalyse
- Drift og sikkerhet
- Infrastructure as Code med Terraform

## Terraform
Terraform-løsningen er bygget modulbasert med egne moduler for:
- Resource Group
- Nettverk
- NSG
- Storage
- Virtuell maskin

## Teknologier
- Microsoft Azure
- Terraform
- Azure Virtual Network
- Azure App Service
- Azure Database for MySQL
- Network Security Groups
- Ubuntu VM
